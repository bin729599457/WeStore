商品接口说明

接口描述 

该接口功能为添加商品接口

接口路径

TOKEN POST /GoodsController/addGoods.do

参数说明

请求参数说明

Body->raw（商品的详情）
 {
 "obj":
 {
 "goods_title":"测试",
 "goods_type_id":"(一级类别ID)",
 "goods_descript":"测试添加书本",
 "goods_price":"0.00",
 "goods_images":"images",
 "goods_nums":"0",
 "goods_author":"测试",
 "goods_publisher":"测试",
 "goods_second_type_id":"（二级类别ID）",
 "is_discounted":"（是否为秒杀商品1为是0为否）"
 }
 }

返回参数说明

名称 类型 备注
t_b_goods t_b_goods 返回新添加的商品对象

接口描述
 该接口功能为搜索商品接口，商品详情即为搜索条件

接口路径

TOKEN POST /GoodsController/selectGoods.do

参数说明

请求参数说明

名称 类型 必填 备注
goods_price_low String no 商品价格范围
goods_price_high String no 商品价格范围
is_discounted String no 是否筛选出秒杀商品（1为是，0或不输入为否）

Body->raw（商品的详情）
 {
 "obj":
 {
 "id":"",
 "goods_title":"",
 "goods_type_id":"",
 "goods_second_type_id":"",
 "goods_descript":"",
 "goods_nums":"",
 "goods_author":"",
 "goods_publisher":""
 }
 }

返回参数说明

名称 备注
goodsList 返回搜索结果的商品详情列表（包括一二级类别名称）

接口描述
 该接口功能为更新商品接口，商品详情即为搜索条件

接口路径

TOKEN POST /GoodsController/UpdateGoods.do

参数说明

请求参数说明

名称 类型 必填 备注
goods_id String yes 要更新的商品ID

Body->raw（商品的详情）
 {
 "obj":
 {
 "goods_title":"nihao",
 "goods_type_id":"10001",
 "goods_descript":"feiyi",
 "goods_price":"36.1",
 "goods_images":"images",
 "goods_nums":"",
 "goods_author":"",
 "goods_publisher":""
 }
 }

返回参数说明

名称 类型 备注
paraMap map 返回更新信息

接口描述
 该接口功能为商品排序接口

接口路径

TOKEN POST /GoodsController/softGoods.do

参数说明

请求参数说明

名称 类型 必填 备注
goods_price String no 根据价格排序(asc升序/desc降序)
goods_sales_nums String no 根据销量排序(asc升序/desc降序)
goods_point String no 根据评分排序(asc升序/desc降序)
goods_weights String no 根据评分排序(asc升序/desc降序)

返回参数说明

名称 类型 备注
goodsList list 返回排序后的商品详情数组

参数说明

请求参数说明

名称 类型 必填 备注
